## Project Overview

Product Store is a full-stack web application built using the MERN stack that allows users to create and manage products with details such as name, price, and image URL. The application features a responsive user interface, light/dark theme support, and seamless integration between frontend and backend services.

---

## Features

- Create new products with name, price, and image URL
- Responsive user interface using Chakra UI
- Light and Dark mode toggle
- RESTful API for product management
- MongoDB Atlas integration
- Real-time form input handling
- Modular and scalable architecture
- Error handling and validation

---

## Technology Stack

### Frontend
- React (Vite)
- Chakra UI
- React Router
- JavaScript (ES6)

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## API Endpoints

| Method | Endpoint             | Description                  |
|--------|----------------------|------------------------------|
| POST   | /api/products       | Create a new product         |
| GET    | /api/products       | Retrieve all products        |
| DELETE | /api/products/:id   | Delete a product by ID       |
